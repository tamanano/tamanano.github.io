薄板样条插值（TPS）

发表于2026-03-01|更新于2026-03-01|Fingerprint-Image-Processing
|阅读量:

主要参考是这里：https://www.geometrictools.com/Documentation/ThinPlateSplines.pdf

薄板样条（Thin-Plate Spline，TPS）是一种根据控制点对图像进行变形的插值方法。TPS是三次样条（Cubic Spline）在多维情况的扩展，针对图像就是二维。其目标是让控制点在尽可能对齐目标位置的同时让弯曲能量最小。

弯曲能量的一般定义为：

E[f]=∫Rn∣D2f∣2 dX\begin{equation}E[f] = \int_{\mathbb{R}^n} | D^2f|^2 \ dX \end{equation} E[f]=∫Rn​∣D2f∣2 dX​​

其中 D2fD^2fD2f 是 fff 的海森矩阵，∣⋅∣2| \cdot |^2∣⋅∣2 是矩阵逐个做平方再相加，也就是F-范数的平方。

所以我们的目标就是找到这个 fff 使得弯曲 E[f]E[f]E[f] 最小 。这和之前我们学的找一组实数的优化问题就不一样了，现在要找的是一个函数。这时候就要引入泛函和变分法来求解这个优化问题。

1. 泛函

泛函定义上来讲是只要映射的值域是标量集合（实数集 R\mathbb{R}R 或复数集 C\mathbb{C}C ）就行，但一般针对泛函的研究其实其定义域就是函数的集合，所以这里给定泛函的定义：给定一个函数空间 F\mathcal{F}F ，泛函 FFF 可以被定义为

F:F→R或F:F→CF:\mathcal{F} \rightarrow \mathbb{R} \quad \text{或} \quad F:\mathcal{F} \rightarrow \mathbb{C} F:F→R或F:F→C

泛函可被写作 F=F[f]F= F[f]F=F[f]。 F\mathcal{F}F 中的函数 fff 被称作容许函数。

2. 变分法

研究求泛函极值的方法被称为变分法。

2.1. 变分法基本引理

变分法建立在变分法基本引理上。一维情况下，变分法基本引理描述为：

成立，则在区间 [x0,x1][x_0,x_1][x0​,x1​] 上，f(x)≡0f(x)\equiv0f(x)≡0。

3. 插值

插值是数值分析领域的内容，已广泛应用于实际应用。常用的插值方法包括最近邻插值、线性插值、多项式插值和样条插值。为过控制点得到平滑的变换，往往采用多项式插值或者样条插值，虽然多项式插值有着更好的平滑性，但控制点较多时多项式插值容易出现龙格现象导致点与点之间的波动过大，所以样条插值常常是最好用的插值方式。

样条插值里有B样条、贝塞尔样条、三次样条等，这之中只有三次样条能保证控制点和目标位置对齐，而我目前的研究的配准是基于特征点的，我们必须保证特征点之间对齐，所以只能选择三次样条。

目标位置 y\boldsymbol{y}y 是 m×1m\times 1m×1 的向量，其元素为 yiy_iyi​，然后系数 a\boldsymbol{a}a 是m×1m\times 1m×1 的向量，其元素为系数 aia_iai​ ，mmm是控制点个数，然后 b\boldsymbol{b}b 是(n+1)×1(n+1)\times 1(n+1)×1 维的向量，元素是系数，bib_ibi​，nnn 是维度数。然后就能把根据控制点解这个样条函数写成线性方程组的形式

3.3 薄板样条插值的正则项

有时候不需要所有的控制点都和目标点对齐，那么可以得到优化目标变成了

E[f]=∑i=1m∣f(xi)−yi∣2+λ∫Rn∣D2f∣2 dX\begin{equation}E[f] = \sum_{i=1}^m | f(x_i)-y_i|^2 + \lambda \int_{\mathbb{R}^n} |D^2f|^2\ dX\end{equation} E[f]=i=1∑m​∣f(xi​)−yi​∣2+λ∫Rn​∣D2f∣2 dX​​

这东西有点像神经网络的损失函数。写成变分法的形式有

E[f]=∫Rn(∑i=1m(f(xi)−yi)2δ(x−xi)+λ∫Rn∣D2f(x)∣2 )dX=∫RnF(f(x),D2f(x))dX\begin{equation}\begin{aligned}E[f] &=\int_{\mathbb{R}^n}\Bigg(\sum_{i=1}^m ( f(x_i)-y_i)^2\delta (x-x_i) + \lambda \int_{\mathbb{R}^n} |D^2f(x)|^2\ \Bigg)dX \\&= \int_{\mathbb{R}^n} F(f(x),D^2f(x)) dX\end{aligned}\end{equation} E[f]​=∫Rn​(i=1∑m​(f(xi​)−yi​)2δ(x−xi​)+λ∫Rn​∣D2f(x)∣2 )dX=∫Rn​F(f(x),D2f(x))dX​​​

根据欧拉-拉格朗日方程可以解出：

∑I=1m(f(xi)−yi)δ(x−xi)+λΔ2f(x)=0\begin{equation}\sum_{I=1}^m (f(x_i)-y_i)\delta(x-x_i) + \lambda\Delta^2f(x)=0\end{equation} I=1∑m​(f(xi​)−yi​)δ(x−xi​)+λΔ2f(x)=0​​

构造样条函数

f(x)=∑i=1mwiG(x,xi)+(b0+∑j=1nbjxj)\begin{equation}f(\boldsymbol{x} ) = \sum_{i=1}^{m} w_i G(\boldsymbol{x},\boldsymbol{x}_i) + (b_0 + \sum_{j=1}^n b_jx_j)\end{equation} f(x)=i=1∑m​wi​G(x,xi​)+(b0​+j=1∑n​bj​xj​)​​

在控制点处：

yj=f(xj)+λwj\begin{equation}y_j = f(\boldsymbol{x}_j)+\lambda w_j\end{equation} yj​=f(xj​)+λwj​​​

写成矩阵形式就变成了

y=(M+λI)w+Nb, NTw=0\begin{equation}\boldsymbol{y} =(M+\lambda I) \boldsymbol{w} +N\boldsymbol{b}, \ N^T\boldsymbol{w} =0\end{equation} y=(M+λI)w+Nb, NTw=0​​

然后解出来就是

w=(M+λI)−1(y−Nb), b=(NT(M+λI)−1N)−1NT(M+λI)−1y\begin{equation}\boldsymbol{w} = (M+\lambda I)^{-1}(\boldsymbol{y} - N\boldsymbol{b}), \ \boldsymbol{b} = (N^T(M+\lambda I)^{-1}N)^{-1} N^T(M+\lambda I)^{-1} \boldsymbol{y}\end{equation} w=(M+λI)−1(y−Nb), b=(NT(M+λI)−1N)−1NT(M+λI)−1y​​
